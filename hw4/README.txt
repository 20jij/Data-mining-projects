This folder consists of:
1. pagerank.py -- the program source code written in Python 3.7.3
2. “graph1.dot” ... “graph5.dot” -- The input files.
3. “pagerank1.csv” ... “pagerank5.csv”-- The output files.
4. CS470_hw4_JasonJi.pdf -- report written in LaTeX
5. latex directory -- LaTeX source code
6. CS470_hw4.pdf -- Instructions
7. this README file

First make sure python3 is installed in your computer. 

Go to the current directory in command line and type in the following command: 

python3 pagerank.py [input_dataset_file_name] [output_file_name] 

ex. python3 pagerank.py graph1.dot pagerank1.csv
