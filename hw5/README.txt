This folder consists of:
1. minHash.java -- the program source code written in Java
2. Inside data folder, “articles_100.truth” ... “articles_10000.truth” -- The input files.
3. Inside data folder, “articles_100.train” ... “articles_10000.train” -- The input files.
4. CS470_hw5_JasonJi.pdf -- report written in LaTeX
5. latex directory -- LaTeX source code
6. CS470_hw5.pdf -- Instructions
7. this README file

First make sure Java is installed in your computer. 

Go to the current directory in command line and type in the following command (which has the default input of articles_1000.train): 

javac minHash.java && java minHash


You can change the input file by changing the numDocs variable in the code.
