This folder consists of:
1. apriori.py -- the program source code written in Python 3.7.3
2. result500.txt -- output file for running the dataset T10I4D100K.txt with minimum support count 500.
3. CS470_hw2.pdf -- report written in LaTeX
4. latex_source_file.tex -- LaTeX source code
5. this readme file

First make sure python3 is installed in your computer. 

Go to the current directory in command line and type in the following command: 

python3 apriori.py [input_dataset_file_name] [min_support] [output_file_name] 

ex. python3 apriori.py T10I4D100K.txt 500 result500.txt
